## 日付をYYYY/MM/DD形式に変換する - formatDisplayDate()

日付をYYYY/MM/DD形式に変換します。

**使用例1**

```javascript
$v.formatDisplayDate(items.DateA);
```

## すべてのコンテキストをログに出力する - allContexts()

すべてのコンテキストをログに出力します。一部のコンテキストは、ログに出力されない場合があります。

**使用例1**

```javascript
$v.allContexts();
```
